Exposes API for managing text in a general screen.

== Description ==

All text called via the API will be registered and displayed under dp-theme-helper Settings.
The text can be altered here and will be updated according in the theme.

== Installation ==

1. Upload `dp-theme-helper.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use `get_theme_text('A section name', 'A field name')` wherever you'd like. Semantic naming would be beneficial.
